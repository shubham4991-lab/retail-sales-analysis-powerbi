# Retail Sales Analysis (Power BI Project)

## Overview
This project analyzes retail transaction data to uncover insights around revenue, returns, customer behavior, and product performance.

## Tools Used
- SQL (data extraction & analysis)
- Python (data cleaning)
- Power BI (dashboard & visualization)

## Key Insights
- Revenue is heavily concentrated in the UK market
- A small group of customers contributes a large share of revenue
- Certain products show disproportionately high return rates
- Seasonal spikes observed during Q4 (Nov–Dec)

## Files
- `01_data_preparation.ipynb` → Data cleaning
- `02_sql_analysis.sql` → SQL analysis
- `03_retail_analytics_dashboard.pbix` → Final dashboard
- `clean_retail.csv` → Clean dataset

## Dashboard Highlights
- Revenue trends over time
- Geographic revenue distribution
- Returns analysis
- Customer segmentation
- Product performance & risk analysis
